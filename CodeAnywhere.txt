🧠 0. What you’re actually doing
Codanywhere = cloud-based IDE (like VS Code running on server)
You are:
•	Creating a remote development environment (container)
•	Writing code
•	Running it in browser
•	Sharing it with others (collaboration)
________________________________________
⚙️ Core Concepts (say these in viva)
Term	Meaning
Container	Your cloud workspace (like remote PC)
IDE	Code editor in browser
Collaboration	Multiple users editing same code
Git	Version control system
________________________________________
🌐 1. Login to Codanywhere
1.	Go to Codanywhere website
2.	Login using:
o	Email / Google / GitHub
________________________________________
🧭 2. Dashboard Overview
You will see options like:
•	Create Container
•	Workspaces
•	Projects
👉 Focus: Create Container
________________________________________
📦 3. Create Container (MOST IMPORTANT STEP)
________________________________________
🔧 Steps:
1.	Click:
👉 Create Container
________________________________________
2.	Choose environment:
•	Node.js OR PHP OR Python
👉 (ANY is fine for exam)
________________________________________
3.	Name:
my-project
________________________________________
4.	Select:
•	Basic server (default)
________________________________________
👉 Click Create
⏳ Wait → container will start
________________________________________
🧠 Think:
Container = your cloud computer
________________________________________
📁 4. Create Project Files
________________________________________
🔧 Steps:
1.	Open File Explorer (left panel)
2.	Right click → New Folder:
MyProject
________________________________________
3.	Inside folder, create files:
•	index.html
•	style.css
•	script.js
________________________________________
✍️ Add basic code:
index.html
<!DOCTYPE html>
<html>
<head>
  <title>My App</title>
</head>
<body>
  <h1>Hello Cloud IDE</h1>
</body>
</html>
